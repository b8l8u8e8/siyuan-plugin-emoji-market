## 🚀 使用方法

1. 在思源中打开表情面板  
   （点击文档的表情按钮，或在文本中输入 `:`）
2. 在搜索框输入关键词  
   两个平台的结果会立即显示在内置结果下方的 **「表情市场」** 区域
3. 点击任意图标，打开预览对话框  
   可查看作者、协议、颜色等信息
4. 按需选择自定义颜色 🎨 或保留原始颜色
5. 勾选确认复选框，点击 **「立即使用」**
6. 图标保存到本地并自动应用 ✅ —— 无需再次点击

🔒 存储说明： 在您确认选择前，表情仅进行动态加载，不会写入本地存储。浏览与预览过程均在内存中完成，关闭即清，不留任何缓存残留。

## 📂 图标存储位置

你选择使用的图标会以 SVG 文件形式保存在思源标准的自定义表情目录中：

- `data/emojis/iconfont/` — 来自阿里巴巴矢量图标库
- `data/emojis/cainiao/` — 来自菜鸟图标

这些都是标准的思源自定义表情文件。  
即使卸载插件也会保留，并在表情面板中正常显示。  

插件本身不存储任何配置或其他数据。
